# ChromeTabManager

一个智能的 Chrome 标签页管理插件，帮助你高效整理浏览器中的标签页。

## 核心功能

*   **读取所有标签页**：全面获取当前浏览器窗口中的所有标签页信息。
*   **智能分组**：支持按域名或类别对标签页进行分组整理。
*   **标签识别**：自动识别陈旧标签页和重复标签页。
*   **一键创建标签组**：快速将整理好的标签页一键应用为 Chrome 标签组。
*   **快速搜索**：支持按标题、域名或 URL 快速查找标签页。
*   **新标签页集成**：打开新标签页时直接进入管理界面。

## 使用方法

1.  打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions/`)。
2.  开启右上角的「开发者模式」。
3.  点击「加载已解压的扩展程序」，选择本项目所在的文件夹即可安装。
